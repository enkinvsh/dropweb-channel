# Дизайн: набор анимированных эмодзи dropweb (ЧЕРНОВИК — ждёт утверждения)

Дата: 2026-06-04
Статус: ждёт утверждения пользователем (стиль залочен, открыто 2 решения)

## Цель
Цельный анимированный набор кастом-эмодзи Telegram, переиспользующий
неоново-фосфорную ASCII/CRT-айдентику dropweb и заменяющий стоковый словарь
эмодзи в постах канала. Паттерн взят у remnawave: фиксированный брендовый
«алфавит» эмодзи в каждом посте + ссылка на пак каждый раз.

## Стиль (залочен брендом)
ASCII/text-mode глифы на CRT-экране: моноширинный (JetBrainsMono), неон на
фоне `#030305`, слоёный gaussian-глоу + сканлайны 4px. Общее «дыхание» блума
связывает набор; у каждого эмодзи свой фирменный моушн = его смысл. Моушн
переиспользует кривую приложения Cubic(0.2,0.8,0.2,1.0)/400мс.

## Набор v1 (16)

## Пайплайн
gpt-image-2 (cliproxyapi) → пост-процесс Pillow (ремап/глоу/сканлайны/кроп 100×100)
→ CRT-аниматор Pillow → ffmpeg vp9 webm.

## Подход: сначала пилот
Собираем ОДНО эмодзи (марка `db`) от и до в реальный `.webm`, проверяем в Telegram,
утверждаем лук, ПОТОМ батчим остальные 15.
