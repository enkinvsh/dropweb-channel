# Публикация набора эмодзи dropweb

1. Сборка: `./run.sh` (или `./run.sh --id db` для пилота). Результат: `build/emoji/*.webm`.
2. Перед заливкой посмотри каждый `.webm` в размере 100×100 (перетащи в «Избранное» в Telegram).
3. Открой бота **@stickers** → `/newemojipack` → формат **видео**.
4. На каждое эмодзи: загрузи `.webm` как ФАЙЛ (не сжатое фото), затем отправь его
   якорный эмодзи (поле `anchor` в `emoji_set.json`, напр. db → 💚, new → 🆕, shield → 🛡).
5. `/publish` → выбери иконку → задай короткое имя → получи `t.me/addemoji/<slug>`.
6. Вшей ссылку в каждый пост канала (как `🌊 t.me/addemoji/...` у remnawave), чтобы
   брендовый словарь расходился и росли установки.

Спека, которую гарантирует сборщик: VP9 webm, 100×100, ≤3с, ≤30fps, ≤256КБ, без звука, зациклено.
