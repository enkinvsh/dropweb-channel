## Как заново вытащить id (если пак пересобран)

`custom_emoji_id` привязаны к КОНКРЕТНОМУ набору. Пересоздал пак — id поменялись, перетяни:

```bash
curl -s "https://api.telegram.org/bot<ТОКЕН>/getStickerSet?name=dropwebpackv1" \
| python3 -c 'import sys,json; \
d=json.load(sys.stdin)["result"]; \
[print(f"{s[\"emoji\"]}\t{s[\"custom_emoji_id\"]}") for s in d["stickers"]]'
```

- Метод `getStickerSet` — read-only, **Premium не нужен**, только валидный токен бота.
- Поле `emoji` у стикера = `anchor` из `emoji_set.json` → так сшивается с внутренними `id`.
- Поле `custom_emoji_id` — то, что подставляется в кнопки/entities.

## Использование на кнопках (Bot API 9.4+)

`InlineKeyboardButton` и `KeyboardButton` получили два поля:

- `icon_custom_emoji_id` — id кастомного эмодзи перед текстом кнопки.
- `style` — цвет кнопки: `"danger"` (красная) / `"success"` (зелёная) / `"primary"` (синяя).

```python
# aiogram (нужна версия с поддержкой Bot API 9.4+)
InlineKeyboardButton(
    text="Кабинет",
    url="https://cab.dropweb.org",
    style="success",                            # зелёная
    icon_custom_emoji_id="5265234995538075373", # on / 🟢
)
```

### Ограничение (важно)

`icon_custom_emoji_id` срабатывает ТОЛЬКО если выполнено одно из:

- у владельца бота есть **Telegram Premium** (для сообщений, которые бот шлёт сам
  в личку/группы/супергруппы), **или**
- бот купил доп. username на **Fragment**.

Иначе поле игнорируется и кнопка рисуется без кастомной иконки.

## Использование в тексте сообщения (entities)

Кастомные эмодзи можно ставить и в текст/подпись через `MessageEntity` типа
`custom_emoji`. У не-Premium пользователей покажется статичный фолбэк по `anchor`.

```python
from aiogram.types import MessageEntity

await bot.send_message(
    chat_id,
    "🟢 Подписка активна",
    entities=[MessageEntity(type="custom_emoji", offset=0, length=2,
                            custom_emoji_id="5265234995538075373")],  # on / 🟢
)
```

`offset`/`length` считаются в UTF-16 (большинство эмодзи = length 2; ZWJ-последовательности
вроде 🏴‍☠️ / 🧑‍💻 длиннее — считай по факту).
